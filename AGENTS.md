# Criterios permanentes de interfaz

## Listas editoriales

- Las filas deben priorizar el contenido: título, fuente, estado y selección.
- No colocar botones repetidos de editar, mover o eliminar dentro de cada fila
  cuando reduzcan el espacio disponible para el título.
- Para acciones sobre elementos, usar selección múltiple mediante casillas y un
  único menú o barra de acciones situado encima de la lista.
- Separar el modo normal de reproducción del modo de edición: las casillas no
  deben aparecer hasta entrar con un botón `Seleccionar` o mediante pulsación
  larga sobre un elemento. El modo edición debe ofrecer `Seleccionar todos` y
  una salida clara como `Hecho`.
- Las acciones por lotes deben conservar el orden relativo de la selección.
- Las actualizaciones y eliminaciones deben mantener la posición de scroll
  siempre que sea posible.
